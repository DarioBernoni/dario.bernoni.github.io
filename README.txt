
# Penalty CV

Mini sito interattivo stile rigori per presentare il tuo CV.

## Come pubblicarlo GRATIS

1. Crea un account GitHub
2. Crea un repository chiamato:
   tuonome.github.io

Esempio:
dariobernoni.github.io

3. Carica questi file nel repository
4. Vai su:
   Settings → Pages
5. Attiva GitHub Pages
6. Il sito sarà online

## Personalizzazioni importanti

Nel file index.html cambia:
- Nome
- Testi
- Link LinkedIn
- Link GitHub
- Link CV PDF

Nel file script.js cambia:
- Messaggi dei rigori
